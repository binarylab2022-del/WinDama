# Code of Conduct

This project welcomes constructive contributions related to Spanish checkers rules, engine development, UI improvements, and research tooling.

Participants are expected to:

- be respectful;
- discuss technical disagreements calmly;
- avoid personal attacks;
- help keep issues and pull requests focused;
- give credit for useful ideas and prior work.

Maintainers may moderate discussions that are abusive, spammy, or unrelated to the project.
