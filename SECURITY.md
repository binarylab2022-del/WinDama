# Security Policy

WinDama is a desktop application and game engine. It does not require network access for normal operation.

## Supported versions

The current open-source preview branch is the supported development branch.

## Reporting a vulnerability

Please report security issues privately through the repository maintainer contact method on GitHub, rather than opening a public issue.

Useful details to include:

- WinDama version or commit hash;
- operating system;
- steps to reproduce;
- affected files or features;
- expected and actual behavior.
