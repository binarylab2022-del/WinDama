# WinDama Screenshot Gallery

This page provides a more complete visual overview of the WinDama application.
The main README intentionally displays only a small selection.

## Main board and engine analysis

![WinDama main board and analysis interface](screenshots/main-board.png)

The primary window combines the playable board, game controls, clock and mode
information, and real-time engine analysis.

## Mandatory capture and continuous analysis

![Mandatory capture highlighting and continuous analysis](screenshots/capture-analysis.png)

Legal destinations are highlighted on the board while the analysis panel
reports the current search.

## Evaluation-weight tuner

![WinDama evaluation weight tuner](screenshots/evaluation-tuner.png)

The tuner supports experimentation with configurable evaluation profiles.

## AI profile tournament

![WinDama profile tournament runner](screenshots/tournament.png)

The tournament tools compare AI or evaluation profiles over multiple games and
record aggregate results.

## Board editor and tactical positions

![WinDama board editor](screenshots/board-editor.png)

The editor creates custom positions for testing, analysis, and tactical
benchmarks.
